README
LAMP Stack Contact Manager by Group 17!

This app aims to be a one size fits all contact manager, with a sleek professional design, suitable for any organization's needs. 

Users can log-in, register, and store as many contacts as they require!
Once logged in, the user can add a contact or search for a contact by their first name, last name, email, or phone number.
These contacts are user specific, and can be edited or deleted as needed.

Passwords must contain a minimum of 8 characters, one letter, one number and one special character.
Inputted names, emails, and phone numbers are also subject to regex validation to ensure inputted data is of correct format.

Technologies used:
- GITHUB: Version control.
- POSTMAN: API testing.
- SWAGGERHUB: API posting and testing.
- APACHE: Web server.
- MYSQL: Database language.
- PHP: API language.
